## 故事的开始 ☕

**無聲咖啡**不只是一杯咖啡。

它是南京的一家社会型企业：
- 咖啡师多为**听障伙伴**，店内用手语和纸条交流
- 每一位员工都在认真生活，用专业能力赢得尊严
- 顾客的每一次消费，都是对他们努力的认可

### 小青的故事
> 听力残疾的她，为了给孩子更好的生活，认真学好每一杯咖啡。  
> 她说不出声音，但咖啡的温度，就是她的语言。

### 阳阳的故事
> 脑瘫伴手部残疾，动作虽慢，但叠袋子、贴LOGO、清洁工作样样认真。  
> 他用自己的节奏，完成每一份工作。
